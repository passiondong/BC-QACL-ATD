# Example data

The files in `demo_data/` are **synthetic** — generated by
`bcqacl_atd.data_io.make_demo_dataset` (or the *Generate synthetic demo data*
button in the wizard). They exist only so the flow, CLI, and GUI run out of the
box. **They are not physically meaningful** and must not be used for real design
or quoted as a performance reference.

To run a real design or reproduce the paper, supply your own data per
[`../docs/data.md`](../docs/data.md):

- transistor `driver_*.s4p`, `power_*.s4p` — single-ended 4-port, 50 Ω;
- load-pull `Z_OPT(f)` `.xlsx`;
- (optional, only for new-technology L_b re-calibration) 27 anchor
  `full_*.s6p` + `half_*.s4p`.

> A publishable set of **real transformer-EM anchors** may be added here in a
> future release. Transistor S-parameters and load-pull data are device/foundry
> data and are typically not redistributable, so they are not shipped.
